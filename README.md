# TaskStudent App

Aplicación sencilla para gestionar tareas de estudiantes.

## Descripción
Este proyecto permite a los estudiantes organizar sus tareas académicas.
Los usuarios pueden agregar tareas, visualizar una lista de pendientes
y marcar tareas como completadas.

## Funciones
- Agregar tareas
- Ver lista de tareas
- Marcar tareas como completadas
